Seletores JQuery

Seletores JQuery são baseados nos seletores CSS existes(id, class, atributos, tipos, valores de atributos, etc)

Alem desses também existem seletores personalizados da propria lib JQuery, um exemplo é o seletor :animated selector 

jQuery(:animated)
.

Para usar qualquer um dos metacaracteres (como !"#$%&'()*+,./:;<=>?@[\]^`{|}~ ) como uma parte literal de um nome, deve ser escapado com duas barras invertidas: \\ Por exemplo, um elemento com id="foo.bar", pode usar o seletor $("#foo\\.bar").
