JQuery

Document Ready

É importante garantir que os elementos que estamos manipulando com JQuery já estejam criados no documento, que já estejam carregados no browser.

Erros comuns ao usar metodos de manipular elementos antes do documento ser totalmente carregado:

  • Tentar esconder um elemento que não foi criado
  • Tentar obter o tamanho de uma imagem que não foi carregada
  • Em resumo tentar acessar/localizar elementos que não foram carregados.
